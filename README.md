# Sriyesh & Anaagha Wedding Invitation Site

Static website clone inspired by `akhila-sarath-wedding.netlify.app`, customized for Sriyesh and Anaagha.

## Files

- `index.html` - page structure and editable text
- `styles.css` - design and responsive styles
- `script.js` - countdown + music toggle
- `assets/hero.jpg` - hero image (you add)
- `assets/music.mp3` - optional music (you add)

## Run Locally

1. Install dependencies:
   - `npm install`
2. Start local server:
   - `npm run dev`
3. Open:
   - `http://localhost:5500`

## Deploy on Netlify

1. Push this project to GitHub.
2. In Netlify, choose **Add new site** -> **Import an existing project**.
3. Select your GitHub repo.
4. Build command: leave empty.
5. Publish directory: `/` (root).
6. Deploy.

Because this is a static site, no framework build step is needed.
